# 8.1 Was ist VPython, und wie funktioniert es im Browser?

Wir haben unsere Kugelbahn digitalisiert, bereinigt und gedruckt. Das physische
Objekt liegt vor uns. Doch wie verhält sich eine Kugel, die diese Bahn
hinunterrollt? Wie schnell ist sie an welcher Stelle? Welchen Einfluss hat die
Neigung auf die Rollzeit? Diese Fragen könnten wir mit einem Stoppuhr-Experiment
beantworten. Wir wählen jedoch einen anderen Weg: Wir bauen ein Computermodell
der Bahn und lassen die Kugel digital rollen. Das ist der Einstieg in die
Simulation.

## Warum reicht Python allein nicht aus?

Stellen wir uns vor, wir berechnen mit Python die Position einer Kugel zum
Zeitpunkt `t = 0.5 s` und erhalten als Ergebnis:

```code
x = 0.32 m, y = 0.12 m, z = 0.04 m
```

Drei Zahlen. Korrekt, aber wenig anschaulich. Wir sehen nicht, wo auf der Bahn
sich die Kugel gerade befindet, wie sie sich bewegt oder ob die berechnete Bahn
überhaupt plausibel aussieht. Genau hier setzt **VPython** an.

VPython ist eine Python-Bibliothek, die physikalische 3D-Szenen in Echtzeit
visualisiert. Wir schreiben Python-Code, und VPython erzeugt daraus ein
interaktives 3D-Fenster. Eine Kugel, die in der Simulation ihren Ort ändert,
bewegt sich auch im Bild. Ein Pfeil, der eine Kraft darstellt, dreht und
streckt sich mit. So können wir Simulationen nicht nur berechnen, sondern
auch beobachten und auf Plausibilität prüfen.

*Warum ist diese visuelle Rückkopplung so wichtig?*

Ein typischer Fehler in physikalischen Simulationen ist ein falsches Vorzeichen
in der Kraftberechnung. Numerisch ist das Ergebnis eine Zahl; ohne
Visualisierung bemerken wir den Fehler vielleicht nicht. Mit VPython sehen
wir sofort, wenn die Kugel nach oben statt nach unten beschleunigt.

In Kapitel 9 werden wir die Physik des Rollens mit VPython modellieren. Dieses
Kapitel legt dafür die Grundlage: Wir lernen die Werkzeuge kennen, bevor wir
sie auf das physikalische Problem anwenden.

## Wie starte ich VPython?

VPython ist in zwei Varianten verfügbar, die denselben Code verstehen:

### Option 1: Web VPython im Browser (kein Setup nötig)

**Web VPython** läuft vollständig im Browser und erfordert keine Installation.
Die Seite [trinket.io/glowscript](https://trinket.io/glowscript) bietet direkt
die Möglichkeit, VPython im Browser auszuprobieren. Alternativ können wir auch
[glowscript.org](https://www.glowscript.org), was allerdings ein google-Konto
erfordert.

```{admonition} Setup: Web VPython
:class: note
1. [glowscript.org](https://www.glowscript.org) aufrufen.
2. Rechts oben auf "Sign in" klicken und ein Google-Konto verwenden.
3. "Create New Program" wählen.
4. Ganz oben steht automatisch `Web VPython 3.x`. Diese Zeile bleibt stehen.
5. Darunter den eigenen Code einfügen und auf "Run" klicken.

Programme lassen sich unter einer öffentlichen URL teilen.
```

### Option 2: vpython im Jupyter Notebook

Für die Projektarbeit verwenden wir VPython direkt im Jupyter Notebook. Wir
importieren es am Anfang jedes Notebooks:

```{code-cell} python
# Dieser Import steht am Anfang jedes Notebooks
from vpython import *
```

```{admonition} Hinweis für macOS- und Linux-Nutzer
:class: note
Das 3D-Fenster öffnet sich als separater Browser-Tab, während das Notebook im
ursprünglichen Tab geöffnet bleibt. Falls sich kein Tab öffnet, einmal
`jupyter nbextension enable --py vpython` im Terminal ausführen und das
Notebook neu starten.
```

## Wie ist das Koordinatensystem aufgebaut?

VPython verwendet ein rechtshändiges kartesisches Koordinatensystem:

- **x** zeigt nach rechts
- **y** zeigt nach oben
- **z** zeigt aus dem Bildschirm heraus (in Richtung der betrachtenden Person)

Das ist das in der Physik und in vielen Ingenieurwerkzeugen übliche System. Für
unsere Kugelbahn bedeutet das: Die Bahn verläuft hauptsächlich in x-Richtung,
die Schwerkraft wirkt in die negative y-Richtung, und die Breite der
Führungsrille erstreckt sich in z-Richtung.

Alle Positionen und Richtungen in VPython werden mit Vektoren angegeben. Ein
**Vektor** fasst drei Zahlen zusammen und wird mit `vec` erstellt:

```{code-cell} python
# Ein Vektor mit x=1, y=0.5, z=0
position = vec(1, 0.5, 0)

# Vektoren können addiert, subtrahiert und skaliert werden:
versatz       = vec(0.2, 0, 0)
neue_position = position + versatz
doppelt       = 2 * position

print(f"Ursprüngliche Position: {position}")
print(f"Neue Position:          {neue_position}")
print(f"Doppelt skaliert:       {doppelt}")
print(f"Betrag (Länge):         {mag(position):.3f}")
```

Die Funktion `mag(v)` gibt den Betrag (die Länge) eines Vektors zurück.

```{admonition} Mini-Übung
:class: tip
Berechnen Sie den Vektor, der von der Position `vec(1, 2, 0)` zur Position
`vec(4, 6, 0)` zeigt. Wie lang ist dieser Vektor?
```

````{admonition} Lösung
:class: tip
:class: dropdown
```python
from vpython import *

start = vec(1, 2, 0)
ende  = vec(4, 6, 0)

differenz = ende - start
laenge    = mag(differenz)

print(f"Richtungsvektor: {differenz}")   # vec(3, 4, 0)
print(f"Betrag:          {laenge:.2f}")  # 5.00
```

Der Vektor `vec(3, 4, 0)` hat den Betrag 5.
````

## Welche Grundobjekte kennt VPython?

VPython stellt eine Reihe vorgefertigter 3D-Objekte bereit. Wir lernen die vier,
die wir in Kapitel 8 bis 10 am häufigsten verwenden.

### sphere: die Kugel

```{code-cell} python
:tags: [skip-execution]
from vpython import *

#kugel = sphere(
#    pos    = vec(0, 0, 0),   # Mittelpunkt
#    radius = 0.05,           # Radius in der gewählten Längeneinheit
#    color  = color.red
#)
```

`pos` ist der Mittelpunkt der Kugel. Das zurückgegebene Objekt `kugel` können
wir später im Code ansprechen, um seine Eigenschaften zu verändern, zum
Beispiel `kugel.pos = vec(0.1, 0, 0)`.

### box: der Quader

```{code-cell} python
:tags: [skip-execution]
from vpython import *

#rampe = box(
#    pos   = vec(0, -0.01, 0),          # Mittelpunkt des Quaders
#    size  = vec(2.0, 0.02, 0.15),      # Ausdehnung in x, y, z
#    color = color.gray(0.6)            # Grauton: 0 = schwarz, 1 = weiß
#)
```

`size` gibt die Ausdehnungen in x-, y- und z-Richtung an. Der Quader ist das
wichtigste Objekt für Flächen, Wände und Führungsschienen.

### cylinder: der Zylinder

```{code-cell} python
:tags: [skip-execution]
from vpython import *

#stab = cylinder(
#    pos    = vec(-0.5, 0, 0),   # Startpunkt
#    axis   = vec(1, 0, 0),      # Richtungsvektor vom Start- zum Endpunkt
#    radius = 0.01,
#    color  = color.blue
#)
```

`axis` beschreibt sowohl die Richtung als auch die Länge des Zylinders. Ein
`axis=vec(1,0,0)` erzeugt einen Zylinder der Länge 1 m entlang der x-Achse.

### arrow: der Pfeil

```{code-cell} python
:tags: [skip-execution]
from vpython import *

#kraft_pfeil = arrow(
#    pos        = vec(0, 0.05, 0),   # Ursprung des Pfeils
#    axis       = vec(0, -0.3, 0),   # Richtung und Länge
#    shaftwidth = 0.015,
#    color      = color.yellow
#)
```

Pfeile eignen sich hervorragend für die Darstellung von Kräften und
Geschwindigkeiten. In Kapitel 9 werden wir mit Pfeilen die Gewichtskraft und die
Normalkraft sichtbar machen.

```{admonition} Mini-Übung
:class: tip
Erstellen Sie eine kleine Szene:

1. Einen grauen Quader an `vec(0, 0, 0)` mit der Größe `vec(1.0, 0.02, 0.2)` –
   das ist unser Rampensegment.
2. Eine rote Kugel an `vec(0, 0.06, 0)` mit Radius `0.05`.
3. Einen gelben Pfeil von `vec(0, 0.06, 0)` mit `axis=vec(0, -0.3, 0)`, der
   die Schwerkraft andeutet.
```

````{admonition} Lösung
:class: tip
:class: dropdown
```python
from vpython import *

scene.background = color.white
scene.title = "Kugel auf Rampe"

rampe = box(
    pos   = vec(0, 0, 0),
    size  = vec(1.0, 0.02, 0.2),
    color = color.gray(0.6)
)

kugel = sphere(
    pos    = vec(0, 0.06, 0),
    radius = 0.05,
    color  = color.red
)

schwerkraft = arrow(
    pos        = vec(0, 0.06, 0),
    axis       = vec(0, -0.3, 0),
    shaftwidth = 0.015,
    color      = color.yellow
)
```

Die Kugel liegt sichtbar über der Rampe (Quader-Oberkante bei y = 0.01,
Kugelmittelpunkt bei y = 0.06, also Abstand von 0.05 m = genau ein Radius).
````

## Wie stelle ich die Szene ein?

Das `scene`-Objekt steuert Kamera, Hintergrund und Titel. Die wichtigsten
Einstellungen legen wir an den Anfang jedes Skripts:

```{code-cell} python
:tags: [skip-execution]
from vpython import *

#scene.title      = "Kugelbahn – Simulation"
#scene.background = color.white
#scene.width      = 800
#scene.height     = 500

# Navigation im 3D-Fenster:
# Linke Maustaste + Ziehen  → Szene drehen
# Scrollrad                 → Zoom
# Rechte Maustaste + Ziehen → Szene verschieben
```

```{admonition} Tipp: Kameraabstand anpassen
:class: note
Wenn Objekte sehr klein erscheinen, hilft `scene.range = 0.5`. Dieser Wert
gibt den Radius des sichtbaren Bereichs in der gewählten Einheit an. Bei
Metern und einer Bahnlänge von 1 m ist `scene.range = 0.8` ein guter
Ausgangswert.
```
