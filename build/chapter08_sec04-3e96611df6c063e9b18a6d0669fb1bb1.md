````{admonition} Übung 8.2 (✩)
:class: tip
**`update()` analysieren**

Betrachten Sie den folgenden Code-Ausschnitt:

```python
from ursina import *

app = Ursina()
kugel = Entity(model='sphere', color=color.blue,
               scale=0.08, position=Vec3(-1.0, 0.05, 0))

GESCHWINDIGKEIT = 0.4   # m/s

def update():
    kugel.x += GESCHWINDIGKEIT * time.dt

app.run()
```

Beantworten Sie ohne den Code auszuführen:

1. In welche Richtung bewegt sich die Kugel?
2. Warum ist `time.dt` hier wichtiger als eine feste Zahl wie `0.01`?
3. Was würde passieren, wenn wir `GESCHWINDIGKEIT = Vec3(0.4, 0, 0)` schreiben
   und `kugel.position += GESCHWINDIGKEIT * time.dt` verwenden?
````

````{admonition} Lösung
:class: tip
:class: dropdown
1. Die Kugel bewegt sich nach rechts (positive x-Richtung) mit 0.4 m/s.

2. Mit `time.dt` ist die Bewegungsgeschwindigkeit unabhängig von der Framerate:
   Auf einem schnellen Rechner (120 fps) wäre `time.dt ≈ 0.008 s`, auf einem
   langsamen (30 fps) `time.dt ≈ 0.033 s`. Multipliziert mit der Geschwindigkeit
   ergibt sich immer derselbe Weg pro Sekunde. Mit einer festen Zahl wie `0.01`
   würde die Kugel auf schnellen Rechnern viel schneller laufen.

3. Das Ergebnis ist identisch: `kugel.x += 0.4 * time.dt` und
   `kugel.position += Vec3(0.4, 0, 0) * time.dt` bewegen die Kugel gleich
   schnell nach rechts. Der Vektor-Ansatz ist flexibler, wenn wir mehrere
   Achsen gleichzeitig bewegen wollen.
````

````{admonition} Übung 8.3 (✩✩)
:class: tip
**Freier Fall visualisieren**

Implementieren Sie in Ursina eine Simulation des freien Falls. Eine Kugel
startet aus der Ruhe bei `y = 1.0` und fällt unter dem Einfluss der
Schwerkraft nach unten. Die Simulation endet, wenn die Kugel den Boden
erreicht (`y ≤ KUGELRADIUS`).

Anforderungen:

1. Erstellen Sie einen Boden (flacher grauer Quader bei `y = 0`).
2. Starten Sie die Kugel aus dem Stillstand bei `y = 1.0`.
3. Verwenden Sie `G = 9.81 m/s²` und das Euler-Cromer-Verfahren.
4. Geben Sie nach dem Aufprall in der Konsole aus: Fallzeit und
   Aufprallgeschwindigkeit.
5. Berechnen Sie die analytischen Werte und vergleichen Sie.
````

````{admonition} Lösung
:class: tip
:class: dropdown
```python
from ursina import *
import math

# --- Anwendung ---
app = Ursina(title='Freier Fall')
window.size  = Vec2(1200, 800)
window.color = color.white

# --- Parameter ---
KUGELRADIUS = 0.04  # [m]
STARTHOEHE  = 1.0   # [m]
G           = 9.81  # Beschleunigung [m/s^2]

# effektive Fallhöhe (von Kugelzentrum bis Bodenberührung)
h_eff = STARTHOEHE - KUGELRADIUS

# Boden
Entity(model='cube', color=color.gray,
       scale=Vec3(1.5, 0.02, 0.5), position=Vec3(0, 0, 0))

# Kugel
kugel = Entity(model='sphere', color=color.red,
               scale=2*KUGELRADIUS,
               position=Vec3(0, STARTHOEHE, 0))

anzeige = Text(text='', position=(-0.5, 0.4), scale=1.0, color=color.black)
EditorCamera()

# --- Anfangszustand ---
v_y        = 0.0    # Start aus der Ruhe
t_sim      = 0.0
laeuft     = True

# --- Animationsschleife ---
def update():
    global v_y, t_sim, laeuft
    if not laeuft:
        return

    # zu große Zeitschritte vermeiden, sonst sieht man die
    # animierte Bewegung nicht mehr
    dt = min(time.dt, 1/120)
    
    # Euler-Cromer: erst Geschwindigkeit, dann Position
    v_y        -= G * dt        # Schwerkraft: v_y wird negativer
    kugel.y    += v_y * dt
    t_sim      += dt

    anzeige.text = f't = {t_sim:.3f} s | v = {abs(v_y):.3f} m/s'

    if kugel.y <= KUGELRADIUS:
        laeuft = False
        kugel.y = KUGELRADIUS  # auf Boden "setzen"

        print('--- Ergebnisse (numerisch) ---')
        print(f'Fallzeit (Simulation):      {t_sim:.4f} s')
        print(f'Aufprallgeschwindigkeit:    {abs(v_y):.4f} m/s')

        t_analytisch = math.sqrt(2 * h_eff / G)
        v_analytisch = G * t_analytisch
        print('--- Ergebnisse (analytisch) ---')
        print(f'Fallzeit (analytisch):      {t_analytisch:.4f} s')
        print(f'Aufprallgeschwindigkeit:    {v_analytisch:.4f} m/s')

app.run()
```
````

````{admonition} Übung 8.4 (✩✩)
:class: tip
**Szene mit Führungsschienen aufbauen**

Bauen Sie eine statische Ursina-Szene, die den Startbereich der Kugelbahn
mit zwei Führungsschienen darstellt:

1. Zwei parallele Schienen aus liegenden Zylindern: bei `z = +0.06` und
   `z = -0.06`, je 1.0 m lang entlang der x-Achse, Durchmesser 0.016 m.
2. Drei Querstreben bei `x = -0.4`, `x = 0`, `x = +0.4`, ebenfalls
   als Zylinder, Durchmesser 0.016 m.
3. Eine rote Kugel mit Radius 0.04 m mittig auf den Schienen.

Es ist keine Animation erforderlich, nur die statische Szene mit `EditorCamera`
soll erstellt werden.
````

````{admonition} Lösung
:class: tip
:class: dropdown
```python
from ursina import *

app = Ursina(title='Kugelbahn mit Führungsschienen')
window.size  = Vec2(1200, 700)
window.color = color.white

# Parameter
SCHIENEN_LAENGE  = 1.0
SCHIENEN_ABSTAND = 0.06          # z-Position der Schienen
DURCHMESSER      = 0.016
SCHIENEN_RADIUS  = DURCHMESSER / 2

KUGELRADIUS      = 0.04

# Hilfsfunktion: Zylinder mit Mittelpunkt im Ursprung, entlang y-Achse
def zylinder_y(radius, hoehe):
    return Cylinder(
        radius = radius,
        height = hoehe,
        start  = -hoehe/2,   # von -h/2 bis +h/2 -> Mittelpunkt bei y = 0
        direction = (0, 1, 0)
    )

# Führungsschienen: entlang x-Achse (y-Zylinder um z-Achse gedreht)
schiene_links = Entity(
    model    = zylinder_y(SCHIENEN_RADIUS, SCHIENEN_LAENGE),
    color    = color.gray,
    rotation = Vec3(0, 0, 90),                 # y -> x
    position = Vec3(0, 0, +SCHIENEN_ABSTAND)
)

schiene_rechts = Entity(
    model    = zylinder_y(SCHIENEN_RADIUS, SCHIENEN_LAENGE),
    color    = color.gray,
    rotation = Vec3(0, 0, 90),
    position = Vec3(0, 0, -SCHIENEN_ABSTAND)
)

# Querstreben: entlang z-Achse (y-Zylinder um x-Achse gedreht)
quer_pos_x = [-0.4, 0.0, 0.4]
for x in quer_pos_x:
    Entity(
        model    = zylinder_y(SCHIENEN_RADIUS, 2 * SCHIENEN_ABSTAND),
        color    = color.gray,
        rotation = Vec3(90, 0, 0),            # y -> z
        position = Vec3(x, 0, 0)
    )

# Kugel mittig auf den Schienen
kugel = Entity(
    model    = 'sphere',
    color    = color.red,
    scale    = 2 * KUGELRADIUS,
    position = Vec3(0, SCHIENEN_RADIUS + KUGELRADIUS, 0)
)

EditorCamera()
app.run()
```
````

````{admonition} Hinweise zum Einstieg
# Diese Zeilen sind Ihr Ausgangspunkt:

**Rotatorisch:**
```python
from ursina import *

app = Ursina(title='Mein Objekt: Rotation')
window.color = color.white

ALPHA       = 90.0    # Winkelbeschleunigung in Grad/s²
ZIEL_WINKEL = 720.0   # Zieldrehung in Grad (720 = zwei Umdrehungen)

objekt = Entity(model='cube', color=color.orange, scale=0.3)
anzeige = Text(text='', position=(-0.85, 0.45), scale=1.2, color=color.black)

winkelgeschw = 0.0       # ω in Grad/s
winkel       = 0.0       # φ in Grad
t_sim        = 0.0
laeuft       = True

def update():
    global winkelgeschw, winkel, t_sim, laeuft
    if not laeuft:
        return

    dt = time.dt

    # Euler-Cromer
    winkelgeschw += ALPHA * dt
    winkel       += winkelgeschw * dt
    t_sim        += dt

    if winkel >= ZIEL_WINKEL:
        winkel = ZIEL_WINKEL
        laeuft = False
        print(f'Zeit für {ZIEL_WINKEL}°: {t_sim:.3f} s')

    objekt.rotation_y = winkel
    anzeige.text      = f't={t_sim:.2f}s | ω={winkelgeschw:.1f}°/s'

app.run()
```
````
