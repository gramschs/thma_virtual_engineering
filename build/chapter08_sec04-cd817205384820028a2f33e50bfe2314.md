````{admonition} Übung 8.2 (✩)
:class: tip
**`update()` analysieren**

Betrachten Sie den folgenden Code-Ausschnitt:

```python
from ursina import *

app = Ursina()
kugel = Entity(model='sphere', color=color.blue,
               scale=0.08, position=Vec3(-1.0, 0.05, 0))

GESCHWINDIGKEIT = 0.4   # m/s

def update():
    kugel.x += GESCHWINDIGKEIT * time.dt

app.run()
```

Beantworten Sie ohne den Code auszuführen:

1. In welche Richtung bewegt sich die Kugel?
2. Warum ist `time.dt` hier wichtiger als eine feste Zahl wie `0.01`?
3. Was würde passieren, wenn wir `GESCHWINDIGKEIT = Vec3(0.4, 0, 0)` schreiben
   und `kugel.position += GESCHWINDIGKEIT * time.dt` verwenden?
````

````{admonition} Lösung
:class: tip
:class: dropdown
1. Die Kugel bewegt sich nach rechts (positive x-Richtung) mit 0.4 m/s.

2. Mit `time.dt` ist die Bewegungsgeschwindigkeit unabhängig von der Framerate:
   Auf einem schnellen Rechner (120 fps) wäre `time.dt ≈ 0.008 s`, auf einem
   langsamen (30 fps) `time.dt ≈ 0.033 s`. Multipliziert mit der Geschwindigkeit
   ergibt sich immer derselbe Weg pro Sekunde. Mit einer festen Zahl wie `0.01`
   würde die Kugel auf schnellen Rechnern viel schneller laufen.

3. Das Ergebnis ist identisch: `kugel.x += 0.4 * time.dt` und
   `kugel.position += Vec3(0.4, 0, 0) * time.dt` bewegen die Kugel gleich
   schnell nach rechts. Der Vektor-Ansatz ist flexibler, wenn wir mehrere
   Achsen gleichzeitig bewegen wollen.
````

````{admonition} Übung 8.3 (✩✩)
:class: tip
**Freier Fall visualisieren**

Implementieren Sie in Ursina eine Simulation des freien Falls. Eine Kugel
startet aus der Ruhe bei `y = 1.0` und fällt unter dem Einfluss der
Schwerkraft nach unten. Die Simulation endet, wenn die Kugel den Boden
erreicht (`y ≤ KUGELRADIUS`).

Anforderungen:

1. Erstellen Sie einen Boden (flacher grauer Quader bei `y = 0`).
2. Starten Sie die Kugel aus dem Stillstand bei `y = 1.0`.
3. Verwenden Sie `G = 9.81 m/s²` und das Euler-Cromer-Verfahren.
4. Geben Sie nach dem Aufprall in der Konsole aus: Fallzeit und
   Aufprallgeschwindigkeit.
5. Berechnen Sie die analytischen Werte und vergleichen Sie.
````

````{admonition} Lösung
:class: tip
:class: dropdown
```python
from ursina import *
import math

app = Ursina(title='Freier Fall', width=900, height=700)
window.color = color.white

KUGELRADIUS = 0.04
STARTHOEHE  = 1.0
G           = 9.81
SPUR_INT    = 0.05

Entity(model='cube', color=color.gray,
       scale=Vec3(1.5, 0.02, 0.5), position=Vec3(0, 0, 0))

kugel = Entity(model='sphere', color=color.orange,
               scale=2*KUGELRADIUS,
               position=Vec3(0, STARTHOEHE, 0))

anzeige = Text(text='', position=(-0.85, 0.45), scale=1.2, color=color.black)
EditorCamera()

v_y        = 0.0
t_sim      = 0.0
spur_timer = 0.0
laeuft     = True

def update():
    global v_y, t_sim, spur_timer, laeuft
    if not laeuft:
        return

    v_y        -= G * time.dt        # Schwerkraft: v_y wird negativer
    kugel.y    += v_y * time.dt
    t_sim      += time.dt
    spur_timer += time.dt

    if spur_timer >= SPUR_INT:
        Entity(model='sphere', color=color.cyan,
               scale=0.015, position=kugel.position)
        spur_timer = 0.0

    anzeige.text = f't = {t_sim:.3f} s | v = {abs(v_y):.3f} m/s'

    if kugel.y <= KUGELRADIUS:
        laeuft = False
        print(f'Fallzeit (Simulation):      {t_sim:.4f} s')
        print(f'Aufprallgeschwindigkeit:    {abs(v_y):.4f} m/s')

        t_ana = math.sqrt(2 * STARTHOEHE / G)
        v_ana = G * t_ana
        print(f'Fallzeit (analytisch):      {t_ana:.4f} s')
        print(f'Aufprallgeschwindigkeit:    {v_ana:.4f} m/s')

app.run()
```
````

````{admonition} Lösung

from ursina import *

app = Ursina(title='Kugelbahn: Startbereich', width=1000, height=600)

SCHIENENLAENGE = 1.0

D_SCHIENE      = 0.016
KUGELRADIUS    = 0.04

Entity(model='sphere', color=color.red,
       scale=2*KUGELRADIUS,
       position=Vec3(0, D_SCHIENE/2 + KUGELRADIUS, 0))
````

````{admonition} Hinweise zum Einstieg
:class: tip
:class: dropdown

# Diese Zeilen sind Ihr Ausgangspunkt:

**Rotatorisch:**
```python
from ursina import *

app = Ursina(title='Mein Objekt: Rotation')
window.color = color.white

ALPHA     = 90.0    # Winkelbeschleunigung in Grad/s²
ZIEL_WINKEL = 720   # Zieldrehung in Grad (720 = zwei Umdrehungen)

objekt = Entity(model='cube', color=color.orange, scale=0.3)
anzeige = Text(text='', position=(-0.85, 0.45), scale=1.2, color=color.black)

winkelgeschw = 0.0
winkel       = 0.0
t_sim        = 0.0
laeuft       = True

def update():
    global winkelgeschw, winkel, t_sim, laeuft
    if not laeuft:
        return
    winkelgeschw      += ALPHA * time.dt
    winkel            += winkelgeschw * time.dt
    objekt.rotation_y  = winkel
    t_sim             += time.dt
    anzeige.text       = f't={t_sim:.2f}s | ω={winkelgeschw:.1f}°/s'
    if winkel >= ZIEL_WINKEL:
        laeuft = False
        print(f'Zeit für {ZIEL_WINKEL}°: {t_sim:.3f} s')

app.run()
```
````

````{admonition} Lösung
:class: tip
:class: dropdown
```python
from ursina import *
import math

app = Ursina(title='Zwei Kugeln im Rennen', width=1200, height=600)
window.color = color.white

G           = 9.81
KUGELRADIUS = 0.04
START_X     = -0.5
ENDE_X      =  0.5
S_EFF       = ENDE_X - START_X
SPUR_INT    = 0.04

WINKEL_1 = 10.0
WINKEL_2 = 25.0
A_1      = G * math.sin(math.radians(WINKEL_1))
A_2      = G * math.sin(math.radians(WINKEL_2))

Entity(model='cube', color=color.gray,
       scale=Vec3(1.2, 0.02, 0.2), position=Vec3(0, 0, 0))

kugel1 = Entity(model='sphere', color=color.red,   scale=2*KUGELRADIUS,
                position=Vec3(START_X, KUGELRADIUS+0.01, 0.04))
kugel2 = Entity(model='sphere', color=color.blue,  scale=2*KUGELRADIUS,
                position=Vec3(START_X, KUGELRADIUS+0.01, -0.04))

anzeige = Text(text='Los!', position=(-0.85, 0.45), scale=1.2, color=color.black)
EditorCamera()

v1, v2                   = 0.0, 0.0
t1, t2                   = 0.0, 0.0
spur1, spur2             = 0.0, 0.0
fertig1, fertig2         = False, False

def update():
    global v1, v2, t1, t2, spur1, spur2, fertig1, fertig2

    if not fertig1:
        v1       += A_1 * time.dt
        kugel1.x += v1 * time.dt
        t1       += time.dt
        spur1    += time.dt
        if spur1 >= SPUR_INT:
            Entity(model='sphere', color=color.red,
                   scale=0.012, position=kugel1.position)
            spur1 = 0.0
        if kugel1.x >= ENDE_X:
            fertig1 = True
            print(f'Kugel 1 (θ={WINKEL_1}°): {t1:.4f} s')

    if not fertig2:
        v2       += A_2 * time.dt
        kugel2.x += v2 * time.dt
        t2       += time.dt
        spur2    += time.dt
        if spur2 >= SPUR_INT:
            Entity(model='sphere', color=color.blue,
                   scale=0.012, position=kugel2.position)
            spur2 = 0.0
        if kugel2.x >= ENDE_X:
            fertig2 = True
            print(f'Kugel 2 (θ={WINKEL_2}°): {t2:.4f} s')

    if fertig1 and fertig2:
        gewinner = 1 if t1 < t2 else 2
        anzeige.text = f'Kugel {gewinner} gewinnt! Δt = {abs(t1-t2):.3f} s'
        print(f'Unterschied: {abs(t1-t2):.4f} s')

app.run()
```

**Analytischer Mittelweg:**
```python
import math
G     = 9.81
S_EFF = 1.0
for w in [10, 25]:
    a = G * math.sin(math.radians(w))
    t = math.sqrt(2 * S_EFF / a)
    print(f'θ={w}°: t={t:.4f}s')

# Für die mittlere Zeit t_m = (t1+t2)/2 gilt:
# sqrt(2·S/a_m) = t_m  → a_m = 2·S/t_m²  → winkel = arcsin(a_m/g)
````
